# Student-Progress-Tracker-
A smart student productivity and performance tracking web application designed to help students manage academic goals, monitor study progress, and improve daily consistency through task management, analytics, and productivity tools.
Student Progress Tracker is a modern productivity-focused web application built to help students organize their academic life efficiently. The platform allows users to track study hours, manage daily tasks, monitor attendance, and visualize progress through interactive dashboards and analytics.

The project focuses on improving discipline, consistency, and academic performance using a clean and user-friendly interface.
->Key Features 
• Daily study progress tracking
• Goal and task management system
• Attendance tracker
• Pomodoro study timer
• Progress analytics dashboard
• Dark and light mode support
• Responsive user interface
• Data storage using browser local storage/database
• Motivational quotes and productivity insights
